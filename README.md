# 🍽️ Omnifood — AI-Powered Food Subscription Website

Omnifood is a modern, responsive web application for a fictional AI-based food subscription service. It showcases a clean UI, smooth user experience, and best practices in frontend development using HTML, CSS, and JavaScript.

---

## 📌 Features

- Fully responsive design (mobile, tablet, desktop)
- Modern and clean UI/UX
- Smooth scrolling navigation
- Sticky navigation bar
- Interactive UI components
- Optimized images and performance
- Pricing plans section
- Testimonials and customer reviews
- Call-to-action (CTA) section
- Accessible and semantic HTML structure

---

## 🛠️ Built With

- **HTML5** — Semantic markup  
- **CSS3** — Flexbox, Grid, animations  
- **JavaScript (ES6)** — Interactivity and DOM manipulation  
- **Google Fonts & Icons**
